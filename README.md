# vue-test-merqueo

## Project setup
```
npm install
```

### Run backend fake REST API with json server 
(located in src/infrastructure/utils and run next command)
```
json-server --watch db.json
```

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
